# Website Audit Issues

## Homepage Issues
1. Title text is WAY too large - overflows and wraps badly, looks unprofessional
2. Chinese characters are enormous and hard to read when they wrap
3. Description text overlaps with the background image - poor readability
4. Bottom trip selector bar looks cramped and unfinished
5. Overall design feels cluttered, not elegant or minimalist
6. Color scheme doesn't match user preference: white, deep blue (#0A1626), gold (#A88B52)
7. The "Supported by OPC Global" footer is barely visible
8. No clear visual hierarchy - everything competes for attention

## User Design Preferences (from knowledge)
- Style: Altruistic, Authentic, Artistic, and Elegant
- Colors: White, Deep Blue (#0A1626), Gold (#A88B52)
- Approach: Minimalist and inspirational
- Font sizes must be consistent, no overflow or overlap
- Mobile responsive

## Trip Detail Page Issues
1. Hero section title text is MASSIVE and wraps horribly - unreadable
2. Left hero section has text overlapping on the image
3. Split layout looks cramped and unbalanced
4. The day images are black and white - looks dull
5. Activity timeline is hard to scan
6. Overall layout lacks elegance and breathing room
7. No consistent color palette applied

## Plan - Complete Redesign
- Redesign BOTH pages from scratch with elegant minimalist approach
- Use color palette: White, Deep Blue (#0A1626), Gold (#A88B52)
- Proper typography: readable sizes, good hierarchy
- Clean layouts with breathing room
- Professional, high-end travel brand feel
- Mobile responsive
